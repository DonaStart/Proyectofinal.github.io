Unidad de Computación: Excel y Páginas HTML

Puntos Buenos

  • Excel facilita el análisis y la visualización de datos.
  • Las páginas HTML permiten la creación de contenido web accesible.
  • Ambas herramientas son fundamentales en el ámbito profesional y académico.
  • Excel ofrece funciones avanzadas para la gestión de grandes volúmenes de datos.
  • HTML es la base de la creación web, esencial para cualquier desarrollador.

Puntos Malos

  • El aprendizaje de Excel puede ser complejo debido a su gran cantidad de funciones.
  • La creación de páginas HTML requiere conocimientos adicionales de CSS y JavaScript para ser realmente efectiva.
  • Excel puede ser propenso a errores si no se utiliza con cuidado.
  • El diseño y desarrollo de páginas web pueden ser tiempo consumibles sin las herramientas adecuadas.
  • Ambas herramientas requieren práctica constante para mantener la habilidad.
